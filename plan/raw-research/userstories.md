**Group E — User Stories** Write at least 5 user stories using this format: *As a \[user\], I want to \[do something\], so that \[reason\].* Cover a range of interactions — spinning, losing, winning big, adding tokens, customizing. Drop them in `raw-research/`.

Example: *As a casual player, I want to see my win history, so that I can feel the highs and lows of the session.*

By: Nick Mitroff, Nikolas Malek

Nick

1. As a hardcore gambler, I want some sort of pity reward for spinning \- after 100 or so  spins, I get a guaranteed payout. That way, I’m always making progress towards something, even on spins that I lose.  
2. As a casual player, I want lots of fanfare \- sound effects, particle animations, screen shake, etc. \- whenever I win something, so that winning on the slots is more exciting.  
3. As a developer, I want separate features to be organized into their own files, so that multiple features can be worked on at once without the risk of merge conflicts.  
4. As a hardcore gambler, I want a choice of multiple different slot machines, so that I can switch to a different slot machine after the one I’m playing on becomes “dry” (meaning that I lose money to it many times in a row).  
5. As a casual player, I want the slot machine to have a creative theme and concept, so that it’s memorable next to all the other online slot machines.  
6. As a hardcore gambler, I want to get a daily login bonus, so that I feel rewarded for my commitment to this slot machine app.

Nikolas 

1. As a casual player, I want an auto spin feature so that I can keep the game running and I don’t have to keep pressing the spin button. (cannot apply to real gambling)  
2. As a frequent player, I want to be able to spend my excess tokens on customizing the slot machine's background or my user avatar, so that I can personalize my gaming experience and show off my wealth.  
3. As a hardcore gambler, I want a global leaderboard showing the biggest daily jackpots, so that I can compete with others and feel like I can win big.  
4. As a casual player, I want the option to watch a short ad when my balance hits zero, so that I can easily add tokens and jump back into playing.  
5. As a cautious player, I want clear, adjustable buttons to raise or lower my bet size before a spin, so that I can manage my risk and make my tokens last longer.  
6. As a social player, I want a share button to appear when I hit a large jackpot, so that I can easily post a screenshot of my big win to my friends on social media.  
7. As a new player, I want to be offered a discounted, one-time starter pack for purchase, so that I feel a strong incentive to make my first in-app purchase.  
8. As a mobile player, I want a quick toggle mute button right on the main game screen so that I can instantly turn off the sounds if I decide to play in a quiet public space.
